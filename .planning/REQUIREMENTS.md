# Requirements: Where Is My Shit (WIMS)

**Defined:** 2026-02-11
**Core Value:** Never lose a conversation again: Instantly recall specific AI discussions or dev sessions.

## v1.3 Requirements (Draft)

Focus: Advanced Retrieval and Search Experience.

### Search Capabilities

- [ ] **SEARCH-01**: User can filter search results by date range (e.g., "last 7 days").
- [ ] **SEARCH-02**: User can filter by source application (Claude, ChatGPT, Cursor).
- [ ] **SEARCH-03**: User can adjust similarity threshold for broader/narrower results.

### User Interface

- [ ] **UI-01**: Search results display rich metadata (source icon, relative time).
- [ ] **UI-02**: "Copy Link" button for deep linking to original context.

## v2 Requirements (Future)

- **INGEST-01**: Generic file ingestion (PDF/MD) via drag-and-drop.
- **INGEST-03**: Audio transcription integration.

## Out of Scope

| Feature | Reason |
|---------|--------|
| Multi-user Auth | Tool is single-user local-first. |
| Cloud Sync | Privacy constraint (v1/v1.2 is local only). |
