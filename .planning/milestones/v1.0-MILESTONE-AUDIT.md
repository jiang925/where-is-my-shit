## Tech Debt & Recommendations

1. **Extension Search**: The browser extension's internal search UI is deprecated and likely broken. Users should be directed to the Phase 4 Web UI.
2. **Authentication**: The API is open on localhost. For v1 (Local-First), this is acceptable, but adding Basic/Token auth is recommended for future releases.
3. **Service Management**: Ensure `wims-watcher.service` is enabled on the host to maintain data flow.
