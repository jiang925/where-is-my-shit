## Overview
Milestone v1.2 focused on radical simplification of the user experience. By replacing the complex JWT authentication flow with a persistent API Key and modernizing the deployment process using `uv`, we made WIMS a "one-command" install and a "zero-maintenance" tool for local AI interaction indexing.
