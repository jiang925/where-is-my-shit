    **1. Create SentenceTransformerProvider** (`src/app/services/providers/sentence_transformer_provider.py`):
    - Class inherits from EmbeddingProvider
    - `__init__(self, model_name: str = "BAAI/bge-m3")`: Instantiate `SentenceTransformer(model_name)` which auto-downloads from HuggingFace on first use. Log model loading progress via structlog. Probe dimensions by encoding a test string during init (same pattern as FastEmbedProvider). Store model, dimensions, model_name as instance vars.
    - `embed(self, texts: list[str]) -> list[list[float]]`: Use `self._model.encode(texts, show_progress_bar=False)`, convert numpy result to list via `.tolist()`. Handle empty list input.

    - Add structlog logger at module level

    **2. Create OnnxProvider** (`src/app/services/providers/onnx_provider.py`):
    - Class inherits from EmbeddingProvider
    - `__init__(self, model_name: str = "BAAI/bge-m3")`: Use `from optimum.onnxruntime import ORTModelForFeatureExtraction` and `from transformers import AutoTokenizer`. Load model with `ORTModelForFeatureExtraction.from_pretrained(model_name, export=True)` and tokenizer with `AutoTokenizer.from_pretrained(model_name)`. Probe dimensions via test encode. Store model, tokenizer, dimensions, model_name.
    - `embed(self, texts: list[str]) -> list[list[float]]`: Tokenize with `self._tokenizer(texts, padding=True, truncation=True, return_tensors="pt")`, run through model, take mean pooling of `last_hidden_state` (masked by attention_mask), convert to list. Handle empty input.
    - Import `torch` for mean pooling calculation
    - `get_dimensions()` and `get_model_name()` return stored values
    - Add structlog logger at module level

    **3. Create ExternalAPIProvider + OpenAICompatibleProvider** (`src/app/services/providers/external_api_provider.py`):
    - `ExternalAPIProvider(EmbeddingProvider)` is an ABC with no additional abstract methods beyond what EmbeddingProvider requires — it serves as a type marker for external/remote providers
    - `OpenAICompatibleProvider(ExternalAPIProvider)`:
      - `__init__(self, model: str, base_url: str, api_key: str | None = None, timeout: int = 30, batch_size: int = 100)`: Create `OpenAI(base_url=base_url, api_key=api_key or "dummy-key", timeout=timeout)`. Store batch_size. Probe dimensions via test embedding. Store model, dimensions.
      - `embed(self, texts: list[str]) -> list[list[float]]`: Process in chunks of self._batch_size. For each chunk call `self._client.embeddings.create(model=self._model, input=chunk)`. Collect and return all embeddings. Handle empty input.
      - `get_dimensions()`, `get_model_name()` return stored values
    - This REPLACES the functionality of OllamaProvider
