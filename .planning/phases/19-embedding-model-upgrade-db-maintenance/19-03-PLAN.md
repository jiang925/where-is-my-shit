    - Test: Explicit fastembed config still works
      - Create `EmbeddingConfig(provider="fastembed", model="BAAI/bge-small-en-v1.5")`, verify FastEmbedProvider created
    - Test: Explicit openai config still works
      - Mock OpenAI client, create config with provider="openai", verify OpenAICompatibleProvider created
    - Test: Provider produces correct dimensions for bge-m3

      - `config = EmbeddingConfig()` → assert config.provider == "sentence-transformers" and config.model == "BAAI/bge-m3"
