# Plan 03-03: Context Extraction Refinement

## Context
The current `wims-watcher` implementation is hardcoded for a single file (`history.jsonl`) and source (`claude-code`). To support future dev tools (Antigravity, Cursor) and improve the quality of the current ingestion, we need to refactor the architecture to be source-agnostic and extract richer metadata from Claude logs.

## Objectives
1.  **Multi-Source Architecture:** Refactor the codebase to support a plugin-like system for different log sources.
2.  **Enhanced Parsing:** Extract `project` context and `pastedContents` from Claude logs.
3.  **Robustness:** Add unit tests for the parser logic.
