# Phase 25 Plan 01: Daemon Distribution System

**Created:** 2026-02-20
**Phase:** 25-daemon-distribution
**Depends on:** Phase 22 (Version Management), Phase 24 (Extension Automation)

## Objective

Create a standalone distribution system for the wims-watcher daemon that enables one-liner installation without requiring git clone. Provide install, update, and uninstall scripts with cross-platform service management (systemd/launchd).

## Success Criteria

1. User can install watcher daemon with `curl -sSL https://raw.githubusercontent.com/jiang925/wims/main/install-watcher.sh | bash`
2. Install script auto-installs uv if not present (with user prompt), then uses uv to manage dependencies
3. Installation automatically sets up systemd (Linux) or launchd (macOS) user service
4. Daemon checks for updates on startup and logs notification to ~/.wims/watcher.log
5. User can manually update with `wims-watcher update` command
6. User can cleanly uninstall daemon with `uninstall-watcher.sh` script

   - Check if installation already exists (idempotent install)

2. **Install uv (with prompt)**
   ```bash

       read -p "uv is not installed. Install it now? (y/n) " -r

           exit 1
       fi
   fi

   - If existing install, preserve ~/.wims/server.json

4. **Install dependencies with uv**
   ```bash
   cd ~/.local/bin/wims-watcher
   uv venv
   uv pip install -r requirements.txt
   ```

5. **Setup service**
   - Detect systemd (Linux) or launchd (macOS)
   - Generate service file from template with correct paths
   - Install to user service directory
   - Enable and start service

6. **Minimal output**
   ```

   ✓ Installed dependencies with uv
   ✓ Configured systemd service

### Task 4: Auto-update Mechanism

**Goal:** Check for updates on startup, notify user in logs, provide manual update command.

**Implementation in `/Users/tianjiang/open-source/where-is-my-shit/wims-watcher/src/updater.py`:**

```python
import json
import logging
import os
import shutil
import subprocess
import sys
import tarfile
from pathlib import Path
from typing import Optional

import requests

GITHUB_API = "https://api.github.com/repos/jiang925/wims/releases/latest"
VERSION_FILE = Path.home() / ".wims" / ".watcher-version"
INSTALL_DIR = Path.home() / ".local" / "bin" / "wims-watcher"


def get_current_version() -> Optional[str]:
    """Read current installed version."""
    if VERSION_FILE.exists():
        return VERSION_FILE.read_text().strip()
    return None


def get_latest_version() -> Optional[str]:
    """Check GitHub for latest release version."""
    try:
        response = requests.get(GITHUB_API, timeout=5)
        if response.status_code == 200:
            data = response.json()
            return data.get("tag_name")
    except Exception as e:
        logging.warning(f"Could not check for updates: {e}")
    return None


def check_for_updates() -> None:
    """Check for updates on startup and log notification."""
    current = get_current_version()
    latest = get_latest_version()

    if latest and current and latest != current:
        logging.info(
            f"Update available: {current} → {latest}. "
            f"Run 'wims-watcher update' to upgrade."
        )


def perform_update() -> bool:
    """Download and install latest version."""
    logging.info("Checking for updates...")

    latest = get_latest_version()
    current = get_current_version()

    if not latest:
        logging.error("Could not fetch latest version from GitHub")
        return False

    if latest == current:
        logging.info(f"Already on latest version: {current}")
        return True

    logging.info(f"Updating {current} → {latest}...")

    try:
        # Download tarball
        response = requests.get(GITHUB_API, timeout=10)
        data = response.json()

        # Find watcher tarball asset
        tarball_url = None
        for asset in data.get("assets", []):
            if "watcher" in asset["name"] and asset["name"].endswith(".tar.gz"):
                tarball_url = asset["browser_download_url"]
                break

        if not tarball_url:
            logging.error("Could not find watcher tarball in release")
            return False

        # Download and extract
        download_path = Path("/tmp/wims-watcher-update.tar.gz")
        extract_path = Path("/tmp/wims-watcher-update")

        logging.info(f"Downloading {tarball_url}...")
        response = requests.get(tarball_url, stream=True, timeout=30)
        with open(download_path, "wb") as f:
            for chunk in response.iter_content(chunk_size=8192):
                f.write(chunk)

        # Extract
        logging.info("Extracting...")
        extract_path.mkdir(exist_ok=True)
        with tarfile.open(download_path, "r:gz") as tar:
            tar.extractall(extract_path)

        # Backup config if exists
        config_file = Path.home() / ".wims" / "server.json"
        config_backup = None
        if config_file.exists():
            config_backup = config_file.read_text()

        # Replace installation
        logging.info("Installing update...")
        shutil.rmtree(INSTALL_DIR)
        shutil.move(str(extract_path / "wims-watcher"), str(INSTALL_DIR))

        # Restore config
        if config_backup:
            config_file.write_text(config_backup)

        # Update version file
        VERSION_FILE.write_text(latest)

        # Restart service
        logging.info("Restarting service...")
        if sys.platform == "darwin":
            subprocess.run(["launchctl", "kickstart", "-k", f"gui/{os.getuid()}/com.wims.watcher"])
        else:
            subprocess.run(["systemctl", "--user", "restart", "wims-watcher.service"])

        logging.info(f"✓ Updated to version {latest}")
        return True

    except Exception as e:
        logging.error(f"Update failed: {e}")
        return False
```

### Task 5: Uninstall Script

**Goal:** Clean uninstall with confirmation prompts.

**File:** `/Users/tianjiang/open-source/where-is-my-shit/wims-watcher/templates/uninstall.sh.template`

```bash
#!/bin/bash
set -e

echo "WIMS Watcher Uninstall"
echo "======================"
echo ""

# Confirmation
read -p "Are you sure you want to uninstall WIMS Watcher? (y/n) " -r
if [[ ! $REPLY =~ ^[Yy]$ ]]; then
    echo "Uninstall cancelled."
    exit 0
fi

echo ""
echo "Uninstalling..."

# Stop and disable service
echo "→ Stopping service..."
if [[ "$OSTYPE" == "darwin"* ]]; then
    launchctl bootout gui/$(id -u)/com.wims.watcher 2>/dev/null || true
    rm -f ~/Library/LaunchAgents/com.wims.watcher.plist
else
    systemctl --user stop wims-watcher.service 2>/dev/null || true
    systemctl --user disable wims-watcher.service 2>/dev/null || true
    rm -f ~/.config/systemd/user/wims-watcher.service
    systemctl --user daemon-reload
fi
echo "✓ Service stopped"

# Remove daemon files
echo "→ Removing daemon files..."
rm -rf ~/.local/bin/wims-watcher
echo "✓ Daemon removed"

# Prompt for config/database removal
echo ""
echo "The following files remain:"
echo "  - ~/.wims/server.json (config file)"
echo "  - ~/.wims/*.lance (database files)"
echo "  - ~/.wims/watcher.log (log file)"
echo ""
read -p "Remove these files? (y/n) " -r
if [[ $REPLY =~ ^[Yy]$ ]]; then
    rm -f ~/.wims/server.json
    rm -f ~/.wims/.watcher-version
    rm -f ~/.wims/watcher.log
    rm -rf ~/.wims/*.lance
    echo "✓ Config and database removed"
else
    echo "Config and database preserved"
fi

echo ""
echo "✓ Uninstall complete"
```

**Goal:** Create release assets that install-watcher.sh can download.

```yaml
  publish-watcher:
    needs: version-sync
    runs-on: ubuntu-latest
    if: github.ref_type == 'tag'
    steps:
      - uses: actions/checkout@v4

      - name: Extract version from tag
        id: version
        run: echo "VERSION=${{ github.ref_name }}" >> $GITHUB_OUTPUT

      - name: Prepare watcher bundle
        run: |
          # Create bundle directory
          mkdir -p watcher-bundle/wims-watcher

          # Copy watcher files
          cp -r wims-watcher/src watcher-bundle/wims-watcher/
          cp wims-watcher/requirements.txt watcher-bundle/wims-watcher/

          # Copy templates
          mkdir -p watcher-bundle/wims-watcher/templates
          cp wims-watcher/templates/* watcher-bundle/wims-watcher/templates/

          # Create version file
          echo "${{ steps.version.outputs.VERSION }}" > watcher-bundle/wims-watcher/VERSION

          # Create tarball
          cd watcher-bundle
          tar -czf ../wims-watcher-${{ steps.version.outputs.VERSION }}.tar.gz wims-watcher/

      - name: Create GitHub Release
        uses: softprops/action-gh-release@v1
        with:
          files: |
            wims-watcher-${{ steps.version.outputs.VERSION }}.tar.gz
            install-watcher.sh
          body: |
            ## WIMS Watcher Daemon ${{ steps.version.outputs.VERSION }}

            Install with:
            ```bash
            curl -sSL https://raw.githubusercontent.com/jiang925/wims/main/install-watcher.sh | bash
            ```

            Update existing installation:
            ```bash
            wims-watcher update
            ```
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
```

   ```

   ```

   ```

4. **Service setup failure**
   ```

   You may need to enable systemd user services:

1. **Already on latest**
   ```

## Risks and Mitigations

1. **Risk: Platform-specific service management bugs**

2. **Risk: Update mechanism fails mid-update**
