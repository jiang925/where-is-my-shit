  <action>
**Create `ui/src/components/CopyablePath.tsx`:**

1. Create an `isFilePath(url: string): boolean` utility function:
   - Returns `true` for Windows absolute paths: regex `/^[A-Za-z]:\\/` (e.g., `C:\Users\...`)
   - Returns `true` for Unix absolute paths: `url.startsWith('/')` (e.g., `/home/user/...`)
   - Returns `false` for URLs starting with `http://` or `https://`
   - Returns `false` for empty/undefined strings
   - For anything else without `://`, treat as path (no protocol = probably a path)

2. Create a `truncateMiddle(path: string, maxLength: number = 60): string` function:
   - If path.length <= maxLength, return as-is
   - Otherwise: frontChars = ceil((maxLength - 3) * 0.4), backChars = floor((maxLength - 3) * 0.6)
   - Return `path.slice(0, frontChars) + '...' + path.slice(-backChars)`
   - The 60/40 split gives more weight to the end (filename) which is more informative

3. Create `CopyablePath` component accepting `{ path: string }` props:
   - Uses `useState<boolean>(false)` for `copied` state
   - `handleCopy` async function: calls `navigator.clipboard.writeText(path)`, sets copied=true, setTimeout 2 seconds to reset to false. Wrap in try-catch, log errors to console.
   - Render a flex container with:
     a. A `<code>` element showing `truncateMiddle(path)` with `title={path}` for full path on hover. Style: `text-xs font-mono text-gray-600 bg-gray-50 px-2 py-1 rounded truncate`
     b. A `<button>` for copy: when `copied` is false, show `Copy` icon (lucide-react) + "Copy Path" text. When `copied` is true, show `Check` icon (lucide-react) + "Copied!" text in green. Style the button similarly to the existing "Open" link: `inline-flex items-center gap-1.5 text-xs font-medium px-3 py-1.5 rounded-full transition-colors`. Use orange theme colors matching claude-code platform config (`bg-orange-50 hover:bg-orange-100 text-orange-700`) for normal state, green (`text-green-600`) for copied state.
   - Export both `CopyablePath` and `isFilePath` (isFilePath needed by ResultCard)

**Update `ui/src/components/ResultCard.tsx`:**

1. Import `{ CopyablePath, isFilePath }` from `./CopyablePath`
2. In the footer section (lines 152-171), replace the current `meta.url` rendering block:

Current code (lines 160-171):
```tsx
{meta.url && (
  <a href={meta.url} ...>Open <ExternalLink /></a>
)}
```

Replace with conditional rendering:
```tsx
{meta.url && (
  isFilePath(meta.url) ? (
    <CopyablePath path={meta.url} />
  ) : (
    <a href={meta.url} target="_blank" rel="noopener noreferrer" ...>
      Open <ExternalLink className="h-3 w-3" />
    </a>
  )
)}
```

Keep all existing styles and behavior for the URL link path. Only change rendering when isFilePath returns true.

Do NOT modify any other part of ResultCard.tsx. The platform badge, time display, content, and conversation ID sections remain unchanged.
