# Plan 06-02: Hardening & CLI Tools

## Goal
Secure the API endpoints, enforce network restrictions, and provide management tools.

## Context
- **Protection:** All content routes need JWT.
- **Network:** Bind 127.0.0.1 by default.
- **Recovery:** CLI tool for password reset.

## Tasks

<task>
<subject>Implement Security Dependency</subject>
<description>
Create the FastAPI dependency for route protection.
</description>
<file_path>src/app/core/security.py</file_path>
<instructions>
Implement `get_current_user(token: str = Depends(oauth2_scheme))`:
- Decode JWT using secret from `AuthDB`.
- Verify `exp` (standard JWT check).
- Verify `iat` > `token_valid_after` (revocation check).
- Raise `HTTPException(401)` on any failure.
</instructions>
</task>

<task>
<subject>Create CLI Management Tool</subject>
<description>
Command-line interface for auth management and server runner.
</description>
<file_path>src/cli.py</file_path>
<instructions>
Create an entry point (e.g., using `typer` or `argparse`):
- `reset-password`:
  - Prompt for new password (or `--auto`).
  - Hash it.
  - Update `AuthDB` with new hash and `token_valid_after = now`.
  - Print confirmation.
- `start`:
  - Run the uvicorn server programmatically.
  - **Enforce bind host**: Default to `127.0.0.1`.
  - Allow override via `--host` flag.
</instructions>
</task>

<task>

</instructions>
</task>
