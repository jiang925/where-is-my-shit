# Plan 06-01: Core Authentication System

## Goal
Implement the foundational security layer: database storage for credentials, password hashing, JWT generation, and the login endpoint.

## Context
- **Strategy:** Self-hosted JWT with SQLite storage.
- **DB:** Dedicated `auth.db` separate from vector data.
- **Hashing:** Argon2 via `passlib`.

<task>
<subject>Implement Auth Database Module</subject>
<description>
Create the SQLite interface for credential storage.
</description>
<file_path>src/app/db/auth.py</file_path>
<instructions>
Create a class `AuthDB` that handles `data/auth.db`.
- **Init:** Ensure table `system_auth` exists.
- **Schema:**
  - `id` (INTEGER PK, check id=1)
  - `password_hash` (TEXT)
  - `token_valid_after` (REAL, timestamp)
  - `jwt_secret` (TEXT)
- **Methods:**
  - `initialize()`: Create table if missing.
  - `get_auth_data()`: Return hash, valid_after, secret.
  - `update_password(hash, valid_after)`: Update credentials.
  - `set_secret(secret)`: Store the persistent JWT secret.
</instructions>
</task>

<task>
<subject>Implement Core Auth Logic</subject>
<description>
Handle hashing and token operations.

<task>
<subject>Create Login Endpoint</subject>
<description>
Expose the login route to generate tokens.
</description>
<file_path>src/app/api/v1/endpoints/auth.py</file_path>
<instructions>
- Create router.
- POST `/login`:
  - Accept JSON body `{"password": "..."}`
  - Verify against `AuthDB`.
  - Return `{"access_token": "...", "token_type": "bearer"}`.
  - Implement progressive delay on failure (time.sleep 1s/2s/4s).
</instructions>
</task>

<task>
<subject>Register Auth Router & Initial Setup</subject>
<description>
Wire up endpoint and handle first-run password generation.
</description>
<file_path>src/app/main.py</file_path>
<instructions>
- Include `auth_router` with prefix `/api/v1/auth`.
- On startup event:
  - Initialize `AuthDB`.
  - Check if `system_auth` is empty (first run).
  - If empty:
    - Generate strong password (using `generate_strong_password()`).
    - Hash and store it.
    - **PRINT THE RAW PASSWORD TO STDOUT** so the user can see it.
    - Print clear instructions: "Initial Admin Password: <password>"
</instructions>
</task>
