# Phase 5 Step 3: CI/CD Pipeline

## Context
With the testing and linting infrastructure in place, we need to automate these checks on every commit to the main branch.

## Tasks
<task_list>
<task>
<description>Create CI Workflow File</description>
<action>
Create `.github/workflows/ci.yml`
</action>
</task>

<task>
<description>Define Backend Job</description>
<action>
Add 'backend-check' job to workflow:
- Runs on ubuntu-latest
- Sets up Python 3.10
- Installs dependencies (`pip install -r requirements.txt`)
- Runs Linting: `ruff check .`
- Runs Formatting Check: `ruff format --check .`
- Runs Tests: `pytest`
</action>
</task>

<task>
<description>Define Frontend Job</description>
<action>
Add 'frontend-check' job to workflow:
- Runs on ubuntu-latest
- Sets up Node.js 18/20
- Working Directory: `./ui`
- Installs dependencies (`npm ci`)
- Runs Linting: `npm run lint`
- Runs Tests: `npm run test -- --run` (ensure single run, not watch mode)
- Runs Build: `npm run build`
</action>
</task>

<task>
<description>Define Extension Job</description>
<action>
Add 'extension-check' job to workflow:
- Runs on ubuntu-latest
- Sets up Node.js 18/20
- Working Directory: `./extension`
- Installs dependencies (`npm ci`)
- Runs Linting: `npm run lint`
- Runs Build: `npm run build`
</action>
</task>
</task_list>
