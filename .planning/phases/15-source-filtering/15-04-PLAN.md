**Integrate into SearchPage.tsx:**

Add PresetButtons between the SourceFilterUI and the main content area. Import PresetButtons and render it:

```tsx
<PresetButtons
  selectedPlatforms={selectedPlatforms}
  onPresetSelect={(platforms) => {
    setSelectedPlatforms(platforms);
    if (platforms.length === 0) {
      searchParams.delete('platforms');
    } else {
      searchParams.set('platforms', platforms.join(','));
    }
    setSearchParams(searchParams);
  }}
/>
```

Place it directly after the SourceFilterUI div, still within the sticky header area.
